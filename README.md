# SQL Generation Assistant

An AI-powered Natural Language to SQL Assistant built using FastAPI, SQLite, and Groq Llama 3.3. The application converts natural language questions into SQL queries, validates them, executes them on a SQLite database, and explains the generated SQL in simple English.

---

## Features

- Convert natural language into SQL using Groq Llama 3.3
- Database schema-aware SQL generation
- Execute SQL queries on SQLite
- SQL validation (Only SELECT queries are allowed)
- SQL explanation in plain English
- REST API built with FastAPI
- Interactive API documentation using Swagger UI
- Structured JSON responses
- Error handling for invalid SQL queries

---

## Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate to the project

```bash
cd sql-generation-assistant
```

Create virtual environment

```bash
python -m venv venv
```

Activate virtual environment

Windows

```bash
venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
GROQ_API_KEY=your_api_key_here
```

Create the sample database

```bash
python create_db.py
```

Run the application

```bash
python -m uvicorn app:app --reload
```

### Query

```
POST /query
```

Example Request

```json
{
    "question": "Show employees from Mumbai"
}
```

Example Response

```json
{
    "success": true,
    "question": "Show employees from Mumbai",
    "generated_sql": "SELECT * FROM employees WHERE city='Mumbai';",
    "explanation": "This query retrieves employees whose city is Mumbai.",
    "results": [
        [1,"Rahul","IT",60000,"Mumbai"],
        [4,"Sneha","IT",55000,"Mumbai"]
    ]
}
```
